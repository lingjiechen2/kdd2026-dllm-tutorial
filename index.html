KDD 2026 Hands-on Tutorial

A Hands-on Tutorial for Diffusion Language Models with dLLM

d LLM

Content

Hands-on diffusion language modeling with dLLM

Focus

This tutorial is centered on the dLLM open-source repository, a unified framework for training, inference, and evaluation of diffusion language models.

Hands-on outcomes

Participants will run interactive inference, adapt models with reproducible recipes, reproduce benchmark results, and extend dLLM with custom samplers or objectives.

Audience

The tutorial is designed for researchers and practitioners who want practical, reproducible access to diffusion language model experiments and tooling.

Schedule

Tutorial Agenda

Module 1
50 min

Foundations and quick demo

Introduce reproducibility challenges, walk through the dLLM repository structure, and run a quick inference demo with an open diffusion language model.

Module 2
25 min

Inference experiments

Vary decoding settings, visualize token generation order, and use dLLM sampler abstractions to compare inference behavior.

Module 3
40 min

Training and open recipes

Cover the unified trainer interface and open recipes such as BERT-Chat, Tiny-A2D, and small-scale adaptation workflows.

Module 4
20 min

Unified evaluation

Run dLLM's unified evaluation pipeline and compare how configuration choices affect reported benchmark numbers.

Module 5
20 min

Framework extension

Implement a minimal scheduler or sampler change and verify it executes inside the existing modular pipeline.

Module 6
25 min

Discussion and troubleshooting

Discuss when diffusion language models are preferable to autoregressive baselines, evaluation sensitivity, tooling gaps, and attendee questions.
